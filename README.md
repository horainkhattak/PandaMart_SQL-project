# Pandamart PostgreSQL Project

## Overview
This project is a PostgreSQL-based analysis of an e-commerce dataset (Pandamart) to extract meaningful business insights from product and sales data.

## Business Insights
- Identified total revenue by category to understand top-performing segments  
- Analyzed out-of-stock products to improve inventory management  
- Detected duplicate products for data quality improvement  
- Evaluated discounts and pricing strategies  
- Calculated price per gram to find best value products  

## Skills Used
- PostgreSQL (SQL Queries)
- Data Cleaning & Transformation
- Aggregation (GROUP BY, HAVING)
- Business Data Analysis

## Objective
To practice real-world SQL data analysis and convert raw data into actionable business insights.
